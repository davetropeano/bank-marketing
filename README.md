# Bank Marketing - Initial Data Analysis

![](1.png)

2. click "New Project"

![](2.png)

3. click "Next"

![](3.png)

4. Create an empty project

![](4.png)

5. Name the project

![](5.png)

6. Click "Add to Project" and select "Notebook"

![](6.png)

![](7.png)

8. The notebook loads. You can scroll through, execute the notebook fresh, etc.
